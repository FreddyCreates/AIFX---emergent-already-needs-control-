Quick Start

Get started with the platform in minutes. First, obtain your API key from the dashboard, then install the SDK for your preferred language.

Installation

Julia 
# Add the ItsNotAILABS registry
using Pkg
Pkg.add("ItsNotAILABS")

# Import the SDK
using ItsNotAILABS

# Initialize with your API key
client = Client(api_key="sk-...")
Python 
# Install via pip
pip install itsnotailabs

# Import and initialize
from itsnotailabs import Client

client = Client(api_key="sk-...")

Your First Request

Make your first API call to the Sovereign Mind model:

Julia 
# Generate text with Sovereign Mind
response = client.generate(
    model = "sovereign-mind-7b",
    prompt = "Explain quantum entanglement in simple terms",
    max_tokens = 256,
    temperature = 0.7

println(response.text)
💡 Pro Tip

Enable phi_optimization = true in your requests to leverage our golden ratio optimization for better coherence and reasoning.

Julia ↔ Motoko Bridge

The Julia-Motoko bridge allows you to execute Julia computations on the Internet Computer's decentralized infrastructure. This is ideal for scientific workflows requiring both high performance and trustless execution.

Basic Usage

Julia 
using MotokoBinding

# Define a computation to run on-chain
@motoko_canister function compute_phi_series(n::Int)
    φ = 1.618033988749
    return [φ^i for i in 1:n]
end

# Call the canister
result = compute_phi_series(10)
println(result)  # Computed on Internet Computer
⚠️ Note

Motoko canisters have cycle costs. Monitor your cycle balance in the dashboard to avoid interruptions.

CPL Reference

CPL (Cognitive Protocol Language) is the orchestration layer for all cognitive functions. Here's a quick reference:

CPL 
# Initialize an organism
init_organism("sovereign-alpha") {
    phi = 1.618033988749
    protocols = ["hebbian", "memory", "attention"]
}

# Load and execute protocols
load_protocol("hebbian-learning")
load_protocol("memory-consolidation")

# Run cognitive cycle
run_cognitive_cycle() {
    attention_threshold = 0.8
    memory_capacity = 1000
}
